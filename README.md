# StudySync Pro

## Project Overview

StudySync Pro is a student productivity dashboard developed using HTML, CSS, and JavaScript. The application helps students organize tasks, manage study sessions, save notes, and track productivity through an interactive and responsive interface.

## Objectives

* Improve student productivity and time management.
* Provide a simple task management system.
* Implement a Pomodoro study timer.
* Store notes and tasks using Local Storage.
* Create a responsive and user-friendly dashboard.

## Features

### Task Manager

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Track total and completed tasks

### Pomodoro Timer

* 25-minute study timer
* Start and reset functionality
* Completion alert notification

### Quick Notes

* Write and save study notes
* Persistent storage using Local Storage

### Productivity Dashboard

* Total Tasks Counter
* Completed Tasks Counter
* Completion Rate Percentage

### Dark Mode

* Toggle between Light and Dark themes
* Theme preference saved locally

### Responsive Design

* Works on desktop, tablet, and mobile devices

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
